# Medical Appointment Management System (FastAPI)

This project is a comprehensive **Healthcare API** developed as part of an Agentic AI/Full-Stack internship. It manages doctors and patient appointments using **FastAPI**, **Pydantic**, and **Python**.

## 🚀 Key Features
The system is divided into three complexity levels:
- **Easy:** Basic CRUD, Pydantic models, and simple fee calculation logic.
- **Medium:** Advanced filtering, status management (Confirm/Cancel), and appointment dependency rules.
- **Hard:** Case-insensitive search, multi-parameter sorting, and pagination logic.

## 🛠️ Tech Stack
- **Framework:** FastAPI
- **Validation:** Pydantic (Python 3.10+)
- **Tools:** Swagger UI (OpenAPI), Git/GitHub

## 📋 Task Outputs (Screenshots)
This repository includes a `screenshots/` folder containing 20 verified outputs for every functional requirement:
1. **Q1-Q5:** Basic Routes & Summaries
2. **Q6-Q9:** Validation & Senior Citizen Discounts
3. **Q10-Q15:** Filtering, Updating, and Status Workflows
4. **Q16-Q20:** Advanced Search, Sort, and Pagination
